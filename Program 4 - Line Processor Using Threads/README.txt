Program: Line Processor Using Threads

Description of Use:
This program creates 4 threads in order to process various lines that are inputed from stdin. Input/output files are also included as part
of the zip file for testing purposes. 

Commands:
$ make
$ ./line_processor
$ ./line_processor < input1.txt > outputtest.txt
$ ./line_processor < input2.txt > outputtest.txt
$ ./line_processor < input3.txt > outputtest.txt
$ make clean
